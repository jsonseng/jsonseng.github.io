Hugo是什么

Hugo是由Go语言实现的静态网站生成器。简单、易用、高效、易扩展、快速部署。

为什么使用Hugo

版本兼容性更好

之前是有用过Jekyll，过了一段时间会有一些升级问题。配置过于复杂，不集中。

使用Hugo，你可以集成多个主题，主题之间的切换，只是在config.yaml / config.toml 里修改主题即可。

这是由于主题的原始文件是存放在themes，而内容与资源文件是有单独的位置存放。做到数据与模版隔离的
